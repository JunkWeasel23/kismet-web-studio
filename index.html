Policies

Add-on Policy:

  • Add-ons are non-refundable once delivered. If removed later, their value becomes a credit for future work.

Refund Policy:

  • Payments are non-refundable once work begins. Add-ons are non-refundable once delivered.

Upgrade Policy:

  • Upgrade anytime by paying the difference between packages. Add-ons carry forward.

Satisfaction Policy:

  • Includes a 1-week revision window for small changes. Major changes are billed as add-ons. Unwanted add-ons become credits.

Source Code and Template Policies:

  • Kismet Web Studio retains full ownership of all internal templates, components, modules, and development systems. Clients receive rights to their finished, customized website-not to the underlying reusable framework.

Source Code Ownership Add-On: $150-300

  • Transfers ownership of project-specific code only. Internal templates remain property of Kismet Web Studio.
